# ADR-022: Use in-app notifications

## Status
Accepted

## Context
Push notifications are ephemeral and can be accidentally swiped away.

## Decision
Use in-app notifications.

## Consequences
Positive: Persistent notification history ensures users don't miss important alerts.
Negative: Needs dedicated UI and unread state management.

## Alternatives Considered
* **Push only**: Rejected due to poor information retention.

## Future Impact
Improves user experience significantly.
