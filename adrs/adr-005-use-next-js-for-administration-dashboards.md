# ADR-005: Use Next.js for administration dashboards

## Status
Accepted

## Context
Administrators and faculty need a robust platform for managing complex workflows, which are unsuited for mobile interfaces.

## Decision
Use Next.js for administration dashboards.

## Consequences
Positive: Administrative workflows are significantly better on desktop than mobile. Next.js provides excellent performance and routing.

## Alternatives Considered
* **React SPA (Vite)**: Next.js provides better out-of-the-box optimizations.

## Future Impact
Provides a dedicated surface area for all complex management tasks.
