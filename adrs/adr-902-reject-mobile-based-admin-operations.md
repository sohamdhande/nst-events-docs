# ADR-902: Reject Mobile-Based Admin Operations

## Status
Rejected

## Context
Organizers requested the ability to manage everything from their phones.

## Decision
Reject Mobile-Based Admin Operations.

## Consequences
Positive: Keeps mobile app simple.
Negative: Administrative workflows belong on web dashboards.

## Alternatives Considered
* **Web Dashboard**: Accepted instead.

## Future Impact
Focuses mobile development efforts on the student experience.
