# ADR-021: Use push notifications

## Status
Accepted

## Context
We need to instantly reach out to students regarding schedule changes or important updates.

## Decision
Use push notifications.

## Consequences
Positive: Drives student engagement effectively.
Negative: Requires user opt-in and careful token management.

## Alternatives Considered
* **Email only**: Rejected because it's too slow.

## Future Impact
Critical for real-time campus communication.
