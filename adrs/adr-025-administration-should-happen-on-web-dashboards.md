# ADR-025: Administration should happen on web dashboards

## Status
Accepted

## Context
Managing complex events and reviewing attendance logs requires screen real estate.

## Decision
Administration should happen on web dashboards.

## Consequences
Positive: Event management and analytics are desktop-oriented workflows.
Negative: Inconvenient for quick mobile edits.

## Alternatives Considered
* **Mobile admin tools**: Rejected (see ADR-902).

## Future Impact
Reduces mobile app bloat.
