# ADR-014: Support Session-Based Attendance

## Status
Accepted

## Context
Some events span multiple days or have distinct sections requiring separate check-ins.

## Decision
Support Session-Based Attendance.

## Consequences
Positive: Accommodates workshops, bootcamps, and multi-day events.
Negative: More complex event creation UI.

## Alternatives Considered
* **Single check-in per event**: Rejected as it fails to track bootcamp retention.

## Future Impact
Provides granular attendance analytics.
