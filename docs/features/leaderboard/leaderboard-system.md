# Merit-Based Leaderboard System

## Status: Frozen
## Version: 1.0
## Architecture State: Accepted

## Core Philosophy
The NST-Events platform adopts a Merit-Based Participation Model. We explicitly reject leaderboards driven merely by event registrations or waitlist placements.

## Individual Leaderboard
Tracks personal cumulative points across all campus activities. Users can view their absolute rank and point breakdowns.

## Club Leaderboard
Club scores are derived organically. Club scores heavily reward:
* Successful Events hosted.
* Major Campus Events executed.
* Competition Achievements of their core members.
* Member Contributions.
**Strict Rule**: Club scores must NOT depend solely on raw member count to prevent spam recruiting.
