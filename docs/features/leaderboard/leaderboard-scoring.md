# Leaderboard Scoring Rules

## The Point Model

### Roles
* `ATTENDEE`: +10
* `VOLUNTEER`: +20
* `ORGANIZER`: +30
* `SPEAKER`: +40
* `MENTOR`: +40
* Session Attendance: +5

### Competition Results
* `PARTICIPANT`: +15
* `TOP_10`: +30
* `SECOND_RUNNER_UP`: +40
* `RUNNER_UP`: +50
* `WINNER`: +75

## Accumulation
Points stack based on the highest role achieved per event. Users cannot receive both ATTENDEE and SPEAKER points for the exact same session.
