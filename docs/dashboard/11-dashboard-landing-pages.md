# Dashboard Landing Pages

## Landing Page Strategy
There is no generic "Home". The landing page is deeply contextualized to the active role.

## Role-Specific Widgets
* **Faculty Mentor**: "Pending Approvals" widget, "My Clubs Summary".
* **Club Admin**: "Upcoming Events", "Active Registrations", "Operations Mode Ready".

## Contextual Home Pages & Quick Actions
Widgets provide 1-click Quick Actions (e.g., "Approve All", "Create Draft") minimizing navigation depth.
