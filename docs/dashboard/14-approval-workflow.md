# Approval Workflow

## Faculty Review & Approval
When an event enters `PENDING_APPROVAL`, it appears in the Faculty Mentor's queue. They can Approve or Reject (with a feedback reason).

## Platform Escalation
Events requiring campus-wide broadcast or special budgets auto-escalate to the `Faculty Admin` or `Platform Admin` queue.

## Feedback Loop & Notifications
Rejections trigger a Notification Center alert to the Club Admin with the provided feedback, reverting the event to `DRAFT`.
