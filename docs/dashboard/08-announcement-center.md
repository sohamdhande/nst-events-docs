# Announcement Center

The Announcement Center handles broadcast human-to-human messaging.

## Announcement Types
* Club Updates
* Event Reminders
* Campus Notices
* Faculty Messages

## Targeting
Announcements can be targeted: "All Students", "Registered Attendees of Event X", or "Club Members".

## Publishing & Delivery Flow
Club Admins draft an announcement. It is queued in PGMQ and delivered as a push notification and an inbox message on the mobile app.

## Analytics
Dashboard provides open rates and delivery success metrics for sent announcements.
