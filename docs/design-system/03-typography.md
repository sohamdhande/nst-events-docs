# Typography

## Brand Typefaces
* **Sans-serif**: `Geist Sans`
* **Monospace**: `Geist Mono`

## Rejected Typefaces
* **Orbitron**: Rejected due to poor legibility at small sizes and an overly "gamer/sci-fi" aesthetic that contradicts our professional operational goals.
* **Inter**: Rejected to establish a more distinct, precise, and modern brand identity via Geist, moving away from generic SaaS defaults.

## Font Hierarchy & Sizes
* `text-xs` (12px): Metadata, secondary badges.
* `text-sm` (14px): Standard UI text, table data.
* `text-base` (16px): Primary body text, inputs.
* `text-lg` (18px): Sub-headers, large buttons.
* `text-xl` (20px): Section headers.
* `text-2xl` (24px): Page headers.

## Weights
* `Regular (400)`: Body text.
* `Medium (500)`: Interactive elements, table headers.
* `Semibold (600)`: Section titles, primary buttons.

## Tracking (Letter Spacing)
* Headings: Tighter tracking (`-0.02em`) for a solid, modern feel.
* Body: Standard tracking.
* Uppercase elements (e.g., tiny badges): Wider tracking (`0.05em`) for legibility.

## Usage Rules
`Geist Mono` is strictly reserved for code snippets, raw data, IDs, and tabular numbers (e.g., live attendance counters) to prevent layout shifting.
