# Leaderboard Components

Reflecting ADR-997 Merit-Based Leaderboard.

## TopPerformerCard
* **Purpose**: Highlighting users with massive merit points.

## AchievementBadge
* **Purpose**: Visual representations of roles achieved (e.g., "Speaker", "Organizer").

## ProgressToNextRank
* **Purpose**: Gamified progress bar motivating continued participation.

## ClubRankingRow
* **Purpose**: Aggregated club scores.

## CompetitionResultsTable
* **Purpose**: Dashboard input surface mapping `user_id` to `WINNER`, `RUNNER_UP`, `TOP_10`, `PARTICIPANT`.
