# Data Visualization Components

## TrendCard
* **Purpose**: KPI + Sparkline (e.g., "Attendance +12%").

## BarChart & DonutChart
* **Purpose**: Breakdown of event demographics or club sizes.

## AttendanceHeatmap
* **Purpose**: GitHub-style activity grid showing user or club engagement over the semester.

## LeaderboardPodium
* **Purpose**: 1st, 2nd, and 3rd place visual display for Competition Results.

## AnalyticsCards
* **Purpose**: Standardized metric wrappers providing consistent padding and export actions.
