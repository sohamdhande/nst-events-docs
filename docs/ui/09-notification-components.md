# Notification Components

## Toast
* **Purpose**: Ephemeral, non-critical success/error messages.

## AlertBanner
* **Purpose**: High-priority system outage or impersonation warnings.

## NotificationTile
* **Purpose**: Actionable items (e.g., "Waitlist Promoted") deep-linking to the exact action screen.

## NotificationDrawer
* **Purpose**: The chronological inbox accessible via the Top Bar.

## BadgeCounter
* **Purpose**: Unread indicator dot on tabs and icons.

## AnnouncementCard
* **Purpose**: Richly formatted broadcast messages with author avatars.
