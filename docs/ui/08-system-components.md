# System Components

The atomic building blocks of the UI.

## Feedback States
* **EmptyState**: Illustration + Copy + Call to Action.
* **LoadingSkeleton**: Animated pulsing shapes matching the layout.
* **ErrorBoundary**: Catches React rendering failures gracefully.
* **PermissionDeniedState**: Strict 403 rendering.
* **OfflineState**: Network connectivity warnings.
* **SuccessState**: Large visual confirmation for critical flows.

## Overlays & Primitives
* **Modal**: Centered dialog requiring explicit interaction.
* **Drawer**: Edge-anchored slide-out panel.
* **Toast**: Ephemeral feedback (auto-dismisses).
* **Banner**: Persistent inline alert.
* **Tooltip**: Hover/Long-press helper text.
* **ConfirmDialog**: Destructive action verification.
