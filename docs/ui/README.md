# UI & Product Surface Documentation

This directory details specific UI component requirements for complex workflows.

## Participant Role Assignment UI
* **Location**: Dashboard -> Event Details -> Registrations Tab.
* **Component**: A dropdown inline with the user row. Allows authorized admins to upgrade an `ATTENDEE` to a `SPEAKER` or `ORGANIZER`.

## Competition Results Submission UI
* **Location**: Dashboard -> Event Details -> Results Tab.
* **Component**: A dedicated form for Club Admins to search participants and assign `WINNER`, `RUNNER_UP`, etc., before marking the event as `ARCHIVED`.

## Competition Results Viewer
* **Location**: Mobile App & Dashboard Event Pages.
* **Component**: A podium/list component displaying verified winners publicly.

## Leaderboard Explanation UI
* **Location**: Mobile App -> Profile -> Leaderboard Info Icon.
* **Component**: A transparent modal explaining the Merit-Based Point Model to students, proving the system's fairness.
