# Component Variants

## Visual Variants (Buttons/Badges)
* **Primary**: Core actions. Uses `primary-base`.
* **Secondary**: Neutral actions. Uses `secondary-base`.
* **Outline**: High-contrast, transparent background.
* **Ghost**: No border, no background (until hover).
* **Destructive**: Uses `error-base`.

## Density Variants (Tables/Lists)
* **Compact**: Minimal padding, small text. (e.g., Audit logs).
* **Default**: Standard operational spacing.
* **Expanded**: Touch-optimized spacing for mobile or accessibility.

## Environment Variants
* **Mobile**: Touch targets forced to 44x44 minimum.
* **Dashboard**: Hover states enabled, dense grids.
* **Operations Mode**: High-contrast, dark-mode forced, massively scaled typography.
