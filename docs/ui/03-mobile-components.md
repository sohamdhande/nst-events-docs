# Mobile Components

These components are built exclusively using NativeWind for Expo.

## Primary Components

### EventCard & FeaturedEventCard
* **Purpose**: Primary discovery surface. Featured variant spans full width.
* **States**: Default, Pressed, Skeleton.

### AttendanceCheckInWidget
* **Purpose**: Priority 1 Home Screen widget.
* **Accessibility**: Supports Tap-To-Confirm fallback.

### AnnouncementTile
* **Purpose**: Feed item for human-to-human broadcasts.

### ProfileHeader
* **Purpose**: High-level personal stats overview.

### NotificationListItem
* **Purpose**: Actionable system alerts (Waitlist promotions).

### LeaderboardRankRow
* **Purpose**: Compact list item showing user rank and merit points.

### ClubIdentityModule
* **Purpose**: Club profile header and follow button.

### SearchOverlay
* **Purpose**: Full-screen modal for FTS event discovery.

### BottomNavigationItem & ActionSheetMenu
* **Purpose**: Core app navigation.

### Carousel
* **Purpose**: Horizontal swiping for upcoming events.
