# Attendance Components

## LocationValidationIndicator
* **Purpose**: Shows whether the user's GPS is within the PostGIS geofence radius.

## DynamicQRCodeRenderer
* **Purpose**: Renders the TOTP HMAC seed on the Dashboard projector. Automatically refreshes every 15 seconds.

## AttendanceStatusPill
* **Purpose**: Color-coded badges (`Present`, `Absent`, `Excused`).

## SessionTimeline
* **Purpose**: Visualizes multi-day or multi-session event check-in points.

## AttendanceOverrideModal
* **Purpose**: Form for capturing the justification when using a Manual Override.

## AttendanceHistoryRow & AttendanceSessionCard
* **Purpose**: Personal profile views.

## AttendanceCountdown
* **Purpose**: "Check-in opens in 15:00".
