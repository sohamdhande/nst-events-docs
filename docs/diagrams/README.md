# Diagram Suite

This directory contains the complete collection of architecture, system, and workflow diagrams for the NST-Connect platform.

All diagrams are generated using [Mermaid.js](https://mermaid.js.org/) syntax for easy version control, rendering, and modification. They act as the visual counterpart to the textual specifications in the rest of the `/docs` directory.
