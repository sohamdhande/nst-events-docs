# Enums

## `event_status`
Tracks the approval and visibility lifecycle of an event.
* `DRAFT`
* `PENDING_APPROVAL`
* `PUBLISHED`
* `ARCHIVED`

## `registration_status`
Manages attendee state and capacity limits.
* `REGISTERED`
* `WAITLISTED`
* `CANCELLED`

## `attendance_status`
The formal record of a user's presence.
* `PRESENT`
* `ABSENT`
* `LATE`

## `event_type`
Categorizes the generic event model.
* `WORKSHOP`
* `HACKATHON`
* `GUEST_LECTURE`
* `COMPETITION`
* `SOCIAL`

## `global_role`
* `PLATFORM_ADMIN`
* `FACULTY_ADMIN`
* `STUDENT`

## `club_role`
* `FACULTY_MENTOR`
