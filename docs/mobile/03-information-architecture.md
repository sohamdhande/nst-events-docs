# Information Architecture

## Top-Level Domains

### 1. Home
Contextual dashboard displaying immediate action items, driven by a single aggregated RPC call.

### 2. Campus
Strictly for **Shared Community Resources**.
* Club Directory
* Campus Discoverability
* Campus Achievements
* Campus Rankings / Top Clubs
* Community Information

### 3. Profile
Strictly for **Personal Record**.
* Attendance History
* Registered Events
* Certificates
* Personal Achievements
* Personal Statistics (My Rank, My Points, My Progress, Personal Position)
* Settings
