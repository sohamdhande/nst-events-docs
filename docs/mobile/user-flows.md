# User Flows

## Registration & Discovery
Users discover events via the Campus tab or direct links. Tapping an event enters the `event-discovery-flow`. Registering triggers the pessimistic `registration-flow`.

## Attendance
If an event is active, the Home Priority System elevates the check-in widget. Tapping it opens the QR Scanner modal.

## Onboarding
New users with empty profiles are routed through the `onboarding-flow` via the Home tab checklist to accelerate time-to-value.
