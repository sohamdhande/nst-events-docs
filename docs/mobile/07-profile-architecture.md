# Profile Architecture

## Profile Governance Rule
The Profile tab is strictly governed. It contains the **Personal Record**.
It must never contain shared community data.

## Included Modules
* **Attendance History**: Verified past check-ins.
* **Registered Events**: Current and past RSVPs.
* **Certificates**: Digital proof of completion.
* **Personal Achievements**: Badges earned.
* **Personal Statistics**: Total hours, workshops attended.
* **Leaderboard (Personal)**: My Rank, My Points, My Progress, Personal Position.
* **Settings**: App preferences and notification toggles.
