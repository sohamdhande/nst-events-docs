# Campus Architecture

## Campus Governance Rule
The Campus tab is strictly governed. It is NOT a miscellaneous dumping ground. It contains **Shared Community Resources**.

## Included Modules
* **Club Directory**: List of all active organizations.
* **Campus Discoverability**: Event feeds and announcements.
* **Campus Achievements**: Global milestones.
* **Community Information**: Broad campus alerts.
* **Leaderboard (Campus)**: Top Clubs, Campus Rankings, Community Leaderboards.\n
